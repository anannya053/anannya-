# Sorry Ankit

Single-page apology website built with plain HTML, CSS, and JavaScript.

## Run it

Open `/Users/anushkatripathi/Documents/New project/index.html` directly in a browser.

If you want to serve it locally instead:

```bash
python3 -m http.server 8000
```

Then open `http://localhost:8000`.

## Personalize it

- Replace the two placeholder image `src` values in `/Users/anushkatripathi/Documents/New project/index.html` with your real photo and Ankit's photo.
- If you want background music, add an `<audio>` element and point it to a small `.mp3` file in the project folder.
